Detailed Description of Nucleus

Nucleus is a platform that makes complex bioinformatics workflows easier and more accessible. It leverages large language models and WebAssembly to automate tasks that usually require advanced computational skills, such as setting up environments, handling diverse datasets, and managing software dependencies.

Core Functionality

Nucleus is a development environment that uses large language models (LLMs) to design and execute bioinformatics workflows. The AI agent within Nucleus can run tools, make real-time adjustments, and respond to human feedback interactively. For example, when analyzing genomic data, Nucleus organizes the necessary steps, executes them, and fixes any issues as they arise, all in real-time. This is made possible by a unified execution environment that lets LLMs run WebAssembly tools directly on a user's machine or server without needing any additional installation. This flexibility makes it easier for biologists to work with a wide range of tools, all within a single, streamlined environment.

Nucleus Prototype

Our product is in early development, but we have completed a proof-of-concept application composed of two main components:

  • Nucleus Extension: An extension for Visual Studio Code providing a chat interface to the LLM agent. Some example queries may be: "identify genetic variants in these files" or "use sed and awk to rename each header in this FASTQ file, then gzip it." Nucleus Extension acquires access to selected user files and passes the query to Nucleus Core. Tool execution is carried out in the extension with the Aioli library.
  • Nucleus Core: The backend of the Nucleus LLM agent. When a query is received from chat, it is sent to an Orchestrator agent prompted with a description of all tools available (each tool is an individual executable WebAssembly program). The orchestrator selects the best tool for the job, then queries a secondary Tool agent. The tool agent constructs a command line input to perform the requested task, assisted by on-the-fly RAG with documentation. When the tool agent outputs a command, it is sent back to the Nucleus Extension to be executed safely on the user's machine.

Biowasm provides 30+ of the most commonly used bioinformatics and Unix tools pre-compiled to WebAssembly. The Nucleus prototype supports all of these tools, and adding new tools is easy.

Looking Forward

We plan to develop Nucleus into a comprehensive integrated development environment (IDE) that not only streamlines bioinformatics workflows but also changes how biologists interact with data. We will integrate visualization capabilities specific to biological data analysis. Nucleus will eventually be a standalone application that runs in a web browser or locally on any system. We aim to create a platform where biologists can not only execute complex analyses with minimal setup but also explore, visualize, and interpret their data in entirely new ways.
