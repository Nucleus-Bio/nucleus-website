Nucleus: Simplifying Bioinformatics Workflows with LLMs and WebAssembly

Nucleus is a platform that makes complex bioinformatics workflows easier and more accessible. It leverages large language models and WebAssembly to automate tasks that usually require advanced computational skills, such as setting up environments, handling diverse datasets, and managing software dependencies.

Nucleus Prototype

  • Nucleus Extension: A Visual Studio Code extension providing a chat interface to the LLM agent, executing bioinformatics tasks directly in the editor with the Aioli library.

  • Nucleus Core: The backend orchestrates tool selection and command generation for executing bioinformatics queries using WebAssembly-based tools.

  • Biowasm Integration: The Nucleus prototype supports 30+ pre-compiled bioinformatics tools, making it easy to add new tools to the platform.
